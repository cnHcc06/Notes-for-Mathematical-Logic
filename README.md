# Notes-for-Mathematical-Logic
## Introduction 
This is the digital notes for the course "Introduction of Mathematical Logic".
The course mainly contains the Propositional Logic (PL) and the First Order Logic (FOL). 
## Content
The note only contains the parts of PL and FOL, but the course also tells some prerequisites such as *logical Connectives*, and some extended knowledges like *Hoare Logic*, which will not be found in the note.   

Nevertheless, these knowledge are either not so hard or not so important, the notes have already show all the crucial things to take a galance at Mathematical Logic or take a test.   

## Document Format
All the note files are in .tmu format, which is dedicated to the formula editor Liii STEM.    
Liii STEM is an efficient formula editor. It uses graphical logic to input formulas and is highly suitable for formula-intensive writing.    
You can download the software from the official website https://liiistem.cn/ ,then you can open these note on it.   
The software also supports exporting pdf, editing pdf,Latex files, etc.
