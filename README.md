# 数理逻辑导论笔记
## 简介 🚀
这是“数理逻辑导论”的课程笔记，  
这门课的主要内容是 **命题逻辑** (Propositional Logic) 和 **一阶谓词逻辑** (First Order Logic)。  
## 笔记内容 📖
1.内容  
笔记也主要包含 **PL** 和 **FOL** 两部分，但这门课还会讲一些前置知识，比如*逻辑关系*，还有一些引申知识，比如*霍尔逻辑*、*程序验证*等。   
笔记中没有包含这些内容，所幸这些内容不是很难，或者并非重点。笔记中的内容已经囊括了数理逻辑导论的核心知识，也和作者的考试范围几乎相同。  

2.语言  
笔记主要使用英语记录，和教学语言相同。  
此外也会穿插一些中文解释和思考。
## 文件格式问题 💾
所有文件均使用 .tmu 格式，这是一款公式编辑器 **Liii STEM** 的专用格式。   
Liii STEM 是一款高效的公式编辑器，它采用图形逻辑输入公式，非常适合公式密集型写作。   

它可以在官网 https://liiistem.cn/ 下载，并且软件内也支持导出pdf、可编辑pdf、Latex文件等多种格式。
